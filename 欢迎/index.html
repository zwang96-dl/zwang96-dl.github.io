👋 欢迎

这是用 GitHub Pages 搭的个人文档库。你现在看到的这篇,就住在 欢迎/index.html ——一个项目一个文件夹,里面一个 index.html,正是你的用法。

怎么加一个新项目?

  1. 在仓库根目录建一个文件夹,名字就是这个项目的 URL,比如 cuda-warp-primitives/(建议用英文/连字符,URL 更干净)。
  2. 把你写的文档存成该文件夹里的 index.html。项目相关的图片等资源也放这个文件夹里。
  3. git add . && git commit -m "新增项目" && git push
  4. 约 1 分钟后主页自动出现这张卡片,访问 你的用户名.github.io/cuda-warp-primitives/ 即可打开。

卡片标题默认就是文件夹名,所以你什么都不加也能用。

想让卡片更精致?(可选)

在项目 index.html 的 <head> 里加这几行:

<meta name="doc-title"       content="展示用的标题">
<meta name="doc-description" content="一句话简介,显示在卡片上">
<meta name="doc-category"    content="推理优化">  <!-- 加了才会出现分类过滤 -->
<meta name="doc-date"        content="2026-07-05">
<meta name="doc-tags"        content="标签1,标签2">

都不加也没关系:标题取文件夹名,日期取 git 提交时间,不显示分类。
