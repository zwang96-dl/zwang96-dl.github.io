快速开始（新用户主线）

支持环境
macOS 14+ · Apple Silicon（arm64）· Python 3.11+ · 8GB 内存 · 仅 CPU 即可（MPS 可选）。核心为纯标准库，无需任何第三方包。 
# 1) 校验离线资源 & 环境自检
python3 course.py verify-offline-bundle
python3 course.py doctor --offline

# 2) 启动本地网页（或直接双击 docs/index.html）
python3 course.py serve

# 3) 运行第一个模拟器，获得第一次成功
python3 course.py run 0
python3 course.py check 0

# 随时问：我在哪、下一步做什么
python3 course.py where-am-i
